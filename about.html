No Trash Search

Google Search is becoming worse as more ad polluted "seo optimized" click farms start taking over search results. This website uses Google's Search tech (which is the best) but only returns results from a whitelisted set of URLs in order to improve the search quality. Note that this site is mostly scientific or engineering-oriented (and definitely not intended for a "broad audience"), but any site with real information is OK.
